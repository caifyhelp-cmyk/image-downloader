# 포트폴리오 이미지 다운로더

포트폴리오/갤러리 사이트의 이미지를 프로젝트별 폴더로 자동 정리해서 다운로드하는 도구입니다.

## 다운로드

👉 [최신 버전 다운로드 (이미지다운로더.exe)](https://github.com/caifyhelp-cmyk/image-downloader/releases/latest)

- 설치 불필요 — 바로 실행
- Windows 10/11 지원

## 사용법

1. `이미지다운로더.exe` 실행
2. **URL** 입력 (포트폴리오 사이트 주소)
3. **저장 폴더** 선택 (기본: 바탕화면/downloaded_images)
4. **▶ 다운로드 시작** 클릭
5. 완료 후 **폴더 열기**로 확인

## 기능

- 포트폴리오 목록 자동 파싱 (더보기 버튼 자동 클릭)
- 각 프로젝트 상세 페이지 순회 → 원본 이미지 전체 수집
- 프로젝트 이름으로 폴더 자동 생성
- 일반 갤러리 사이트도 지원 (img 태그 전체 수집)
- 새 버전 자동 감지 및 1클릭 업데이트

## 지원 사이트 예시

| 사이트 유형 | 동작 |
|------------|------|
| 참공간 등 포트폴리오 CMS | 프로젝트별 폴더 + 전체 이미지 |
| 일반 갤러리 페이지 | 페이지 내 이미지 전체 다운로드 |

## 업데이트

새 버전이 나오면 앱 상단에 **업데이트 있음** 버튼이 표시됩니다.  
클릭하면 자동으로 다운로드 후 재시작됩니다.
